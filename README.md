# hamburger-order
햄버거 주문 App을 만들어 보자!

```
- 식사 유형(포장/매장)에 따른 선택
- 메뉴(햄버거, 사이드, 음료) 선택 후 장바구니 담기
- 원하는 옵션 추가 가능
```
